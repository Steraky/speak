# speak

To use speak you wil need:

```
Python3.x
pip3
git
```

To install speak you will do this :

(Linux)
```
git clone https://github.com/Steraky/speak
```
```
cd speak
```
```
pip3 install requierements.txt
```
```
python3 speak.pyc
```

(Windows)
```
cd speak
```
```
pip install requierements.txt
```
```
python speak.pyc
```
